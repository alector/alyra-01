# XOR

> The exclusive disjunction (XOR) outputs true only when inputs differ, if one is true and the other is false.

| Input A | Input B | Output |
| ------- | :-----: | -----: |
| false   |  false  |  false |
| false   |  true   |   true |
| true    |  false  |   true |
| true    |  true   |  false |
